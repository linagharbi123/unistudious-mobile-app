Politique de confidentialité pour Unistudious

Date d'entrée en vigueur : 21 novembre 2024

Dernière mise à jour : 21 novembre 2024

Bienvenue chez Unistudious ! Votre vie privée est importante pour nous. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles lorsque vous utilisez notre application mobile.

1. Informations que nous collectons

Nous collectons les types d'informations suivants :

1.1 Informations personnelles

  • Identifiants : Nom d'utilisateur, identifiant unique de compte
  • Informations de contact : Adresse e-mail, numéro de téléphone (optionnel)
  • Informations de profil : Nom complet, photo de profil, date de naissance, lieu de naissance, genre, adresse
  • Informations d'authentification : Tokens d'authentification (Google, Facebook, Apple Sign In)

1.2 Contenu utilisateur

  • Photos : Images téléchargées pour les cours, messages et profil
  • Vidéos : Vidéos téléchargées pour les cours et messages
  • Messages texte : Messages privés et de groupe via le chat
  • Fichiers : Documents PDF et autres fichiers partagés
  • Enregistrements audio : Messages vocaux et enregistrements de cours

1.3 Données d'utilisation

  • Interactions avec l'app : Pages visitées, fonctionnalités utilisées
  • Notifications push : Préférences de notification, historique des notifications
  • Données de session : Sessions actives, temps d'utilisation

1.4 Informations techniques

  • Informations sur l'appareil : Type d'appareil, système d'exploitation, version de l'application
  • Identifiants d'appareil : Identifiants uniques pour les notifications push
  • Données de diagnostic : Rapports de crash et erreurs (via Firebase)

1.5 Informations de localisation

  • Adresse IP : Collectée automatiquement lors de l'utilisation de l'app
  • Localisation géographique approximative : Dérivée de l'adresse IP (non précise)

2. Comment nous utilisons vos informations

Vos informations sont utilisées pour :

  • Fournir et améliorer nos services d'apprentissage en ligne
  • Gérer votre compte et authentification
  • Faciliter la communication entre étudiants et éducateurs
  • Envoyer des notifications push concernant les cours, messages et mises à jour
  • Personnaliser votre expérience d'apprentissage
  • Assurer la sécurité et prévenir la fraude
  • Analyser l'utilisation de l'application pour améliorer nos services

3. Partage de vos informations

Nous ne vendons pas vos informations personnelles. Cependant, nous pouvons partager vos données avec :

3.1 Fournisseurs de services tiers

  • Google (Firebase) : Pour les notifications push, l'authentification et l'analyse
  • Google Sign-In : Pour l'authentification via compte Google
  • Facebook : Pour l'authentification via compte Facebook
  • Apple : Pour l'authentification via Sign in with Apple
  • Rocket.Chat : Pour les services de messagerie et chat en temps réel
  • Jitsi Meet : Pour les réunions vidéo et sessions en ligne

3.2 Autorités légales

Nous pouvons divulguer vos informations lorsque cela est requis par la loi ou pour faire respecter nos termes et politiques.

3.3 Transferts d'entreprise

En cas de fusion, d'acquisition ou de vente d'actifs, vos informations peuvent être transférées.

4. Vos droits en matière de confidentialité

Vous avez le droit de :

  • Accéder : Demander une copie de vos données personnelles
  • Corriger : Modifier vos informations personnelles via les paramètres de l'app
  • Supprimer : Demander la suppression de votre compte et de vos données
  • Vous désinscrire : Désactiver les notifications push dans les paramètres
  • Restreindre : Limiter le traitement de vos données
  • Portabilité : Demander le transfert de vos données

5. Sécurité des données

Nous mettons en place des mesures de sécurité robustes pour protéger vos données :

  • Chiffrement SSL/TLS pour la transmission des données
  • Stockage sécurisé des mots de passe et tokens d'authentification
  • Accès restreint aux informations sensibles
  • Audits de sécurité réguliers

Note importante : Aucun système n'est totalement sécurisé. Nous vous encourageons à protéger vos identifiants de compte et à ne jamais les partager.

6. Conservation des données

Nous conservons vos données aussi longtemps que nécessaire pour :

  • Remplir les objectifs décrits dans cette politique
  • Respecter les obligations légales
  • Résoudre les litiges et faire respecter nos accords

Lorsque vous supprimez votre compte, nous supprimons vos données personnelles dans un délai raisonnable, sauf si la conservation est requise par la loi.

7. Cookies et technologies de suivi

Notre application mobile utilise des technologies similaires aux cookies pour :

  • Maintenir votre session connectée
  • Analyser l'utilisation de l'application
  • Améliorer les performances et la fonctionnalité

Nous n'utilisons pas ces technologies pour le suivi publicitaire entre applications.

8. Données collectées par des tiers

Les services tiers suivants peuvent collecter des données lorsque vous utilisez notre application :

  • Firebase (Google) : Collecte des données d'utilisation et de diagnostic pour améliorer les services
  • Rocket.Chat : Collecte les messages et métadonnées de communication
  • Jitsi Meet : Collecte les données de session vidéo (audio/vidéo non enregistrés par défaut)

Ces services ont leurs propres politiques de confidentialité que nous vous encourageons à consulter.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité périodiquement. Les modifications seront publiées sur cette page avec la date de "Dernière mise à jour". Nous vous informerons des changements importants via l'application ou par e-mail.

10. Protection des mineurs

Notre application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans. Si vous êtes parent ou tuteur et que vous pensez que votre enfant nous a fourni des informations personnelles, contactez-nous immédiatement.
